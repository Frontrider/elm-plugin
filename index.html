elm-plugin

Elm language plugin for IntelliJ IDEA.

Features

  • Supports Elm 0.17.0
  • Parsing the syntax
  • Syntax highlighting and color settings page
  • Going to declaration
  • Highlighting unresolved references
  • Code completion
  • Spellchecking
  • Brace matching
  • Rename refactoring

Releases

The official JetBrains' repository should contain the plugin that supports the newest version of Elm. If you need to have different Elm version supported then download one of the packages below, use the "Install plugin from disk..." option and do not upgrade the plugin from the JetBrains' repository even when a newer version exists.
